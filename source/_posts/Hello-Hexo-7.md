## 环境

- Hexo: 7.3.0
- hexo-theme-next: 8.27.0
- Node.js: 18.15.0

## 常用命令

新建文章：

```bash
hexo new "文章标题"
```

本地预览：

```bash
hexo server
```

部署到 GitHub Pages：

```bash
hexo clean && hexo deploy
```
